### 需求

高度固定，100px，左边栏宽度200px，右边栏宽度300px，中间栏宽度自动填满。

### 方案
#### 1、float 
注意事项:center必须在文档流中的最后一个     
优点：      
缺点：center元素的宽度实际是100%宽度，只是被浮动元素撑开，表面上看满足需求，实际不然        
高度自适应：不能        
兼容性：高      
#### 2、absolute
注意事项:       
优点：      
缺点：      
高度自适应：不能        
兼容性：高      
#### 3、flex
注意事项:       
优点：      
缺点：      
高度自适应：能      
兼容性：IE8不兼容       
#### 4、table
注意事项: table默认是高度自适应，如果需要固定高度，需要做额外处理       
优点：      
缺点：      
高度自适应：默认自适应      
兼容性：高      
#### 5、grid
注意事项:   
优点：  
缺点：  
高度自适应：能  
兼容性：IE8不兼容   
#### 6、calc
注意事项:
优点：  
缺点：  
高度自适应：不能    
兼容性：IE8不兼容
